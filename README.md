# About

PytoPaint is a software for cluster-based analysis of flow cytometry data.

# Installation:

This program can be bundled into an executable using PyInstaller.
1. Download and install [uv](https://docs.astral.sh/uv/getting-started/installation/).
2. Clone the repository.
3. Run `uv sync` to configure the local environment.
4. Run `uv run pyinstaller PytoPaint.spec --noconfirm` to build the executable.
5. The executable can be found in the newly created `dist` directory.
